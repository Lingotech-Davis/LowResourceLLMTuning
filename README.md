# LowResourceLLMTuning
Project to demonstrate fine-tuning methods large models for low resource languages.

Organization:
* `practice_notebooks`: Notebooks we used for training our models.

Models:
* Cantonese-only ASR model: [chengyili2005/whisper-small-canto](https://huggingface.co/chengyili2005/whisper-small-canto)
* Mandarin -> Cantonese ASR model: [chengyili2005/whisper-small-mando-canto](https://huggingface.co/chengyili2005/whisper-small-mando-canto)
